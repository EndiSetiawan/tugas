#Part 1#__Melacak Lokasi Secara Akurat Dengan Seeker di Kali Linux__


Artikel ini bakal ngebahas trik keren abis nih! Gue mau share cara nge-track lokasi HP pake alat canggih banget, namanya Seeker di Kali Linux. Nah, ini bukan main-main, loh! Cuma dengan kirim link ke korban, kita bisa tau lokasinya dengan akurat. Triknya? Pake phising, bro! Phising itu kayak senjata andalan hacker buat bikin link berbahaya yang bisa nyolong data kita. Gila, kan? Stay tuned, guys, biar nggak ketinggalan!


1. Install Tools Seeker

Langkah pertama adalah dengan melakukan cloning repositori dari github:

git clone http: https://github.com/thewhiteh4t/seeker.git





cd seeker && ls





python3 seeker.py


Jika terdapat kegagalan atau server dalam keadaan offline, anda bisa menjalankan perintah dibawah ini untuk memulai tools Seeker secara manual, anda dapat menjalankan perintah berikut :

python3 seeker.py -t manual




2. Menjalankan Tools Seeker

Sekarang kita akan menjalankan tools seeker ini untuk mendapatkan lokasi korban.Sebelum itu download Ngroknya di:.>>Klik Disni dan ikuti langkahnya

1.Login Ngrok untuk mendapatkan token.
2.Tempel Authoken di terminal.
3.Masuk ke directory tempat Ngrok di download dengan mengetik cd ..? contoh:cd usr/local/bin
4.setelah masuk, jalankan Ngrok dengan perintah .ngrok http ?nomor port




karena kita memakai port 80 maka kia akan pakai perintah berikut:

./ngrok http 80



Disini saya akan menggunakan metode penyerangan menggunakan link undangan grup WhatsApp. Saya akan menginput nomor 2, setelah itu saya menginput nama grup “Info Sunmori” dan berikutnya saya memasukkan foto yang sudah didownloads. Setelah di enter secara otomatis akan dibuatkan server khusus oleh ngrok yang akan berbentuk sebuah link.



-------------------------------------------------

Sekarang berlanjut ke ngrok. Disini anda harus menjalankan perintah berikut untuk dapat mengaktifkan server ngrok:


./ngrok http 80


Ketika menjalankan perintah tersebut akan muncul halaman dari ngrok itu sendiri. Terdapat beberapa informasi yang dapat kalian lihat di gambar. Dapat anda lihat terdapat link pada Forwading seperti https://23cf-198-199-118-42.ngrok.io, itulah link yang akan dikirim kepada korban. Salin link tersebut dan kirimkan kepada korban/target.





Seperti ini tampilan ketika korban membuka link yang kita kirim. Jelas jika yang membukanya orang awam, pasti akan memencet tombol Join Chat. Padahal jika memencet tombol tersebut, semua informasi lokasi anda akan terekspose sampai koordinat dan lokasi kota tempat anda tinggal berbahaya sekali bukan?.




Ketika memencet tombol “Join Chat” akan memunculkan pesan meminta izin lokasi seperti gambar dibawah ini. Jika korban mengizinkan lokasi, maka secara otomatis data lokasi anda semuanya secara cepat akan dikirimkan kepada pengirim link tersebut.




Seperti inilah contoh jika lokasi anda terlacak. Mulai dari informasi device yang anda gunakan, Informasi lokasi seperti kota, negara, koordinat lokasi saat ini bahkan IP kita bisa diketahui. Lebih parahnya terdapat link Gmaps yang menunjukan lokasi target secara akurat!.
